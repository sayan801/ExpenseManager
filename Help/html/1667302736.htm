DataRect Structure

Describes a width, a height and location of a rectangle. This type is very similar to [System.Windows.Rect], but has one important difference: while [System.Windows.Rect] describes a rectangle in screen coordinates, where y axis points to bottom (that's why [System.Windows.Rect]'s Bottom property is greater than Top). This type describes rectange in usual coordinates, and y axis point to top.
